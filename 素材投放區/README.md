# 素材投放區 — 圖片抽換說明

想換網站上的圖，**只要做兩步**：

1. 把新圖丟進對應資料夾（覆蓋規則看下面的檔名）
2. 雙擊專案根目錄的「**同步圖片.bat**」（或跟 Claude 說「同步圖片」）

腳本會自動：壓縮成網頁規格 → 覆蓋上線檔 → 推上 GitHub。約 3-13 分鐘後全球生效。
**網頁完全不用改**，原始圖會自動備份到 `_已處理\`。

---

## 📁 作品集\（精選作品，共 12 格）

- 檔名 = 作品編號：`01.jpg` ~ `12.jpg`（jpg/png 都可以）
- 編號 = 顯示順序：01 在最前面。想調整順序就改檔名重丟
- 首頁會顯示其中 6 張（01、02、04、06、10、11），完整 12 張在精選作品頁

## 📁 HERO\（各頁面頂部大圖）

檔名 = 頁面名稱：

| 檔名 | 對應頁面 |
|---|---|
| `about.jpg` | 關於 CHUN.EN |
| `a-voir.jpg` | À Voir |
| `b-room.jpg` | B.room |
| `c-photos.jpg` | C.photos |
| `signature.jpg` | 商業視覺定位企劃 |
| `booking.jpg` | 預約與時程 |
| `enterprise.jpg` | 企業合作 |

（首頁頂部是影片，不在此機制內；要換影片跟 Claude 說）

---

## ⚠️ 注意事項

- **檔名絕對不可出現客戶姓名**（隱私規則），一律用編號
- 原圖越大越好（建議寬 2000px 以上），腳本會自動壓縮
- 換作品圖後，圖片的替代說明文字（alt）可能與新圖不符——換圖後跟 Claude 說一聲「更新 alt」即可
- 換圖後手機看到舊圖 = 快取，等 10 分鐘或重新整理
