## A｜À Voir：形象顧問與未來形象規劃

A 主要提供形象顧問服務，核心是協助客戶打造「未來形象」。

我們主要做的是商務視覺，服務對象以商務人士為主。舉例來說，今天如果我要跟一群老闆吃飯，而我也希望自己未來能成為其中的一員，當大家都穿著符合場合與角色的服裝，只有我穿著 T-shirt，就可能顯得格格不入。

所以，我們會根據客戶目前的職業類別，以及他未來想前往的狀態，例如晉升中高階主管、準備創業、成為品牌主理人或開始經營個人品牌，協助他建立符合下一階段角色的專屬形象。

這項服務最大的特色，是從每一個人的個人特質、職業屬性與未來目標出發，而不是把所有被認為「好看」或「高級」的元素，全部強加在客戶身上。

我們不是把客戶塑造成同一種樣子，而是找出最適合他的專業表達方式。

## CHUN.EN 與一般形象照拍攝的差異

很多人會問，我們和一般形象照拍攝有什麼不同。

第一個差別，是我們會協助客戶進行服裝選品。

很多人平常不太會嘗試新的服裝選擇，也不知道自己適合什麼、在不同職業角色中應該穿什麼。因此，我們會根據客戶的個人風格、身形特色、職業調性與實際使用場合，協助他進行服裝選擇。

不少客戶在過程中會發現，原來自己可以有不一樣的樣子，也會在服裝嘗試、妝髮設計與拍攝過程中，逐步累積自信。

另一個很大的差別，是我們希望客戶的線上與線下形象能夠保持一致。

一般拍攝形象照時，有些人可能會從衣櫃裡拿出一套平常幾乎不會穿的西裝，拍攝結束後再將它收回去。那套形象只存在於拍攝當天，像是上戲與下戲的狀態。

線上看到的是一套正式、精緻的形象照，但實際見面時，卻可能感受到很大的落差。

CHUN.EN 希望做到的，是讓客戶在官網、社群、名片與形象照片中呈現的樣子，也能延續到實際工作、公開場合與日常接觸之中。

所以，我們不只做整體造型規劃，也會進一步思考整體應用。

## 以終為始的影像規劃

很多人拍完形象照之後，才發現照片不好用、不知道怎麼使用，或真正要製作官網、電子名片、LINE 圖文選單與宣傳海報時，才發現缺少適合的素材。

因此，我們採取「以終為始」的規劃方式。

在拍攝之前，我們會先了解客戶未來可能使用影像的場景，例如：

- 官網
- 社群平台
- 電子名片
- LINE 圖文選單
- 簡報
- 活動海報
- 媒體採訪
- 講師介紹
- 個人品牌內容

確認使用需求後，再反向規劃服裝、妝髮、構圖、背景、橫直式比例與拍攝素材。

這樣在正式拍攝時，才能一次取得足夠且適合後續應用的內容，避免拍完之後才發現某些照片無法裁切、某些版位缺少素材，或不同平台之間無法延伸使用。

## CHUN.EN：串聯 A、B、C 的整合型服務

我們目前最主要推動的服務，是將 A、B、C 三項專業串聯在一起。

過去，客戶要拍攝一組形象照，可能需要自己分別尋找：

- 形象顧問
- 服裝或選品協助
- 彩妝師
- 髮型師
- 攝影師
- 攝影棚
- 後續設計與應用人員

而 CHUN.EN 希望提供的是由同一套策略統籌的整合服務。

目前場域中也分別規劃了：

- 形象顧問區
- 妝髮造型區
- 攝影區

我們會先了解客戶的職業類別、現階段需求、未來目標、預算與影像用途，再推薦適合的服務內容。

在執行範圍上，可能依照客戶需求規劃：

- 一組妝髮與服裝造型
- 兩組妝髮與服裝造型
- 三組妝髮與服裝造型

也可以依後續需求，延伸製作電子名片、實體名片、LINE 圖文選單、社群素材或其他品牌應用內容。

# 四、「未來形象」概念

「未來形象」是目前最有辨識度的核心概念之一，但需要定義得更精準。

若直接表達成「想成為老闆，所以要穿得像老闆」，容易被誤解為：

- 模仿成功者
- 假裝擁有某個身分
- 只追求外表升級
- 用服裝掩飾能力不足
- 把所有人塑造成同一種商務樣貌

CHUN.EN 真正要表達的應該是：

> **不是模仿某一種成功者的樣子，而是讓你的外在表達，逐步與下一階段的職業角色對齊。**

可延伸為：

> **形象不能取代能力，但能讓能力更快被正確理解。**

以及：

> **當你的能力已經向前走，形象也不應停留在上一個階段。**

---

## 建議定位方式

目前不建議直接把某個套餐命名為「未來形象方案」。

較穩定的做法是：

- **品牌方法**：未來角色形象定位
- **顧問服務**：商務形象定位企劃
- **整合服務**：商業視覺整合企劃
- **對外主張**：從你現在的位置出發，為下一階段的角色建立一致、可信且能實際使用的專業形象

如此「未來形象」會成為 CHUN.EN 的獨特觀點，而不是容易被誤解的套餐名稱。

# 五、真正差異不是「一站式」，而是「同一套策略」

「一站式」可以保留，但不適合作為最主要賣點。

許多攝影棚也能提供：

- 攝影
- 妝髮
- 場地
- 服裝
- 修圖

如果只說「我們通通包辦」，市場辨識度仍然有限。

CHUN.EN 真正的差異應該是：

> **不是把三項服務綁在一起，而是讓三項專業都從同一個定位出發。**

更直接地說：

> **一般的一站式服務，解決的是不用分別找人的麻煩；CHUN.EN 的整合服務，解決的是每一位執行者各做各的問題。**

# 七、不建議用「一妝、兩妝、三妝」作為主要方案邏輯

「一組、兩組、三組妝髮造型」可以作為內部估價與執行規格，但不適合成為高端服務的第一層商品邏輯。

客戶真正購買的不是「換幾次妝」，而是：

- 我要呈現幾種角色？
- 我要面對哪些使用情境？
- 我需要多少類型的素材？
- 這些素材要使用在哪裡？

建議先定義**視覺情境**，再決定妝髮與服裝套數。

| 視覺情境 | 可能用途 | 對應造型 |
|---|---|---|
| 專業權威 | 官網、媒體、簡報 | 正式服裝與俐落妝髮 |
| 親和溝通 | 社群、講師介紹、自媒體 | 較柔和、可接近的造型 |
| 品牌主理人 | 品牌故事、專訪、形象影片 | 具個人辨識度的服裝與場景 |

所以不應先問客戶：

> 要一妝、兩妝，還是三妝？

而應先問：

> 這次影像需要支援哪些角色與使用情境？

再由 CHUN.EN 判斷需要幾組服裝、妝髮與影像配置。

# 八、建議調整的對外用語

| 目前口述說法 | 建議對外說法 |
|---|---|
| 想成為老闆的樣子 | 讓形象與下一階段的職業角色對齊 |
| 不要顯得格格不入 | 在重要場合中呈現與角色相符的專業感 |
| 外面拍形象照 | 多數形象照服務以拍攝執行為起點 |
| 拍照像演戲 | 形象只在鏡頭前成立，未延伸到真實工作場景 |
| 線下是蓬頭垢面的業務 | 線上影像與實際接觸產生明顯落差 |
| 通通包辦 | 由同一套策略統籌整合 |
| 相關攝影通通都有做 | 聚焦人物、品牌與商業內容影像 |
| 形象改造 | 形象定位、形象升級、未來角色企劃 |
| 一套妝裝法 | 一組妝髮與服裝造型／一組視覺情境 |
| 一站式套組 | 商業視覺整合企劃 |

「B 就像一般彩妝工作室」與「C 的攝影通通有做」可以作為內部理解，但不適合作為正式品牌文案。

前者會降低 B.room 的專業層級；後者則會讓 C.photos 看起來什麼都接，反而失去人物、品牌與商業影像的聚焦。

# 十五、À Voir 核心商品定義

## 15.1 Entry｜未來角色形象診斷

### 服務定位

一項低門檻、具專業交付的付費入口。

它不是銷售諮詢，也不是只聊感受，而是協助客戶理解：

- 現有外在表達與職業角色是否一致
- 下一階段角色需要什麼視覺語言
- 接下來應優先調整哪些部分

### 適合對象

- 即將晉升或轉職
- 正準備創業
- 開始經營個人品牌
- 專業能力已提升，但形象仍停留在前一階段
- 不確定自己適合什麼風格
- 不確定是否需要完整企劃
- 想先取得專業方向，再決定後續投入

### 可能包含

- 現況形象盤點
- 現職與未來角色分析
- 個人特質辨識
- 風格關鍵字
- 服裝與妝髮方向
- 重要場合需求
- 下一階段執行順序

### 必須具備的交付

至少需有一項可被客戶保留與使用的成果，例如：

- 診斷摘要
- 風格關鍵字頁
- 未來角色視覺方向頁
- 優先調整清單
- 後續行動建議

### 服務邊界

診斷負責：

> **告訴客戶方向。**

診斷不等於：

- 完整衣櫥重整
- 完整選品清單
- 實體陪購
- 正式妝髮設計
- 拍攝企劃
- 品牌應用製作

## 15.2 Core｜商務形象定位企劃

### 服務定位

À Voir 的核心主力商品。

它的目的不是只提出建議，而是建立一套可被實際使用的個人形象系統。

### 核心內容

- 現況與未來角色定位
- 職業屬性與利害關係人分析
- 個人特質與辨識度
- 商務場合需求
- 衣櫥盤點
- 服裝組合
- 色彩、輪廓與材質方向

- 線上與線下形象一致性

### 服務成果

核心企劃負責：

> **陪客戶建立可以真正執行的形象系統。**

它與診斷的差異不是只有時間長短，而是成果深度：

| 比較項目 | 未來角色形象診斷 | 商務形象定位企劃 |
|---|---|---|
| 核心目的 | 釐清方向 | 建立完整可執行系統 |
| 服務深度 | 分析與建議 | 分析、規劃與實際落地 |

| 交付 | 診斷摘要與行動建議 | 完整形象策略與執行地圖 |

# 十九、品牌應用模組

品牌應用需區分兩種完全不同的服務：

## 19.1 應用規劃

協助客戶確認：

- 照片與影片要使用在哪裡
- 需要哪些橫直式比例
- 需要哪些構圖留白
- 需要多少素材類型
- 哪些影像應用於官網、社群、簡報、名片與媒體
- 拍攝前應預留哪些版位與使用情境

此項可作為整合企劃的一部分。

---

## 19.2 設計製作

實際完成：

- 電子名片
- 實體名片
- LINE 圖文選單
- 社群版型
- 活動與演講海報
- 官網人物素材
- 簡報封面
- 媒體資料包
- 其他品牌應用

此項涉及獨立設計、製作、修改與交付，不應只用「包含應用」概括，也不應默認全部包含在拍攝方案中。

# 二十、方案命名與造型套數原則

## 20.1 不以「一妝、兩妝、三妝」作為第一層商品邏輯

「一組、兩組、三組妝髮造型」可作為：

- 內部估價
- 現場執行規格
- 拍攝工時拆解
- 攝影流程配置

但不適合作為高端整合服務的主要商品名稱。

客戶真正購買的是：

- 需要呈現幾種角色
- 需要支援哪些使用情境
- 需要多少類型的素材
- 影像要在哪些平台使用

因此，應先定義**視覺情境**，再決定服裝、妝髮與拍攝套數。

| 視覺情境 | 可能用途 | 對應造型 |
|---|---|---|
| 專業權威 | 官網、媒體、簡報 | 正式服裝與俐落妝髮 |
| 親和溝通 | 社群、講師介紹、自媒體 | 柔和、具接近感的造型 |
| 品牌主理人 | 品牌故事、專訪、形象影片 | 具辨識度的服裝與場景 |
| 日常專業 | 顧客接觸、工作現場、社群內容 | 可延續至日常的實穿造型 |

對客戶的第一個問題不應是：

> 要一妝、兩妝，還是三妝？

而應是：

> 這次影像需要支援哪些角色與使用情境？

---

## 20.2 暫不使用 Base／Pro／Premium 作為官網主架構

Base／Pro／Premium 可保留於：

- 內部銷售工具
- 個別提案
- 訪談後方案比較
- LINE 一對一報價

在服務邊界、交付與價格尚未完整核准前，不應同時與其他分類系統出現在官網。

# 二十一、LINE Bot 與知識庫同步原則

既有 LINE Bot 曾以 A／B 表示：

- A＝單項直接預約
- B＝諮詢型服務

但現在 A、B、C 已正式代表三個子品牌：

- A＝À Voir
- B＝B.room
- C＝C.photos

因此，V2 正式取消 A 類／B 類作為客服分流名稱。

改為：

- **直接預約型服務**
- **需求評估型服務**
- **整合企劃型服務**
- **企業專案型服務**

### 建議分流

```text
直接預約型
├─ 妝髮
├─ 眉型
├─ 已標準化的單項造型
└─ 已標準化的單項攝影

需求評估型
├─ 未來角色形象診斷
├─ 商務形象定位企劃
├─ 衣櫥與選品
└─ 尚未確認服務方向的客戶

整合企劃型
├─ CHUN.EN 商業視覺整合企劃
├─ 平面＋動態整合
└─ 後端品牌應用

企業專案型
├─ 團隊形象
├─ 高階主管影像
├─ 企業妝髮與拍攝
└─ 年度或大型專案
```

# 二十二、價格與方案建立順序

本版不沿用尚未核准的舊方案與數字。

正式建立價格前，應依以下順序完成：

> **服務邊界 → 交付內容 → 執行流程 → 工時 → 人力與外包成本 → 修改與售後 → 方案組合 → 價格 → 官網與 LINE**

每一項服務至少需先確認：

- 適合對象
- 解決的問題
- 服務內容
- 不包含內容
- 服務時長
- 會議與現場次數
- 團隊配置
- 交付格式
- 交付時間
- 修改次數
- 地點與交通
- 訂金、改期與取消
- 是否可折抵或升級
- 成本與合理毛利

# 二十五、V2 最終定義

CHUN.EN 不再被描述為：

> 形象顧問＋妝髮工作室＋攝影棚的一站式套組。

V2 正式定義為：

> **CHUN.EN 是一個以未來角色形象定位為起點，整合形象策略、服裝選品、妝髮造型、平面與動態影像，再延伸至品牌實際應用的商業視覺整合系統。**

品牌核心差異：

> **我們不是把多個服務綁在一起，而是讓每一項專業，都從同一個定位出發。**

品牌服務成果：

> **讓客戶在線上與線下，都呈現一致、可信，且足以支持下一階段角色發展的專業形象。**
