## 🟡 強烈建議(上線後第一個月)

### 5. 客戶實際作品 ≥ 30 張
| 分類 | 預期張數 | 說明 |
|---|---|---|
| **À Voir 形象規劃** | 5-10 組 | 形象前後對比 / 衣櫃整理紀錄 / 風格座標說明圖 |
| **B Room 妝髮造型** | 10-15 張 | 依場合分:婚禮 / 形象照 / 活動 / 自媒體 — 最少每類 3 張 |
| **C Photos 形象攝影** | 15-20 張 | 個人形象 / 企業形象 / 品牌攝影 |

| 規格 | |
|---|---|
| 解析度 | 長邊 1800-2400px (壓縮後 < 500KB / 張) |
| 格式 | JPG q85 或 WebP |
| 色調 | 統一精品克制風(Aesop / Le Labo / Brunello Cucinelli 視覺參考) |
| 授權 | **必須取得客戶肖像權同意書**;不便露面者用側面/半身/局部 |

### 6. Hero Loop Video
| 用途 | index.html hero 區塊(目前是文字 hero,加 loop video 大幅提升質感) |
|---|---|
| 內容 | 服務側拍特寫:化妝刷掃過 / 翻動衣架 / 按下快門 / 整理衣領 / 試妝光影 |
| 長度 | 10-15 秒(無聲循環) |
| 規格 | 1920×1080 或直幅 1080×1920 / MP4 H.264 / **檔案 < 4MB**(web 載入) |
| 取代物 | 取代目前的「巨型 C 字」靜態 hero |
| 投資回報率 | 高 — 高端品牌(Aesop, Le Labo, 北歐 studios)目前的標配 |

### 7. 服務進行中紀錄照
| 用途 | 各 hero、各服務頁、portfolio 補充 |
|---|---|
| 內容 | 諮詢中(俯拍長桌)/ 試妝中(手部 + 鏡子)/ 拍攝中(攝影師 + 模特剪影)/ 試衣中(衣架 + 手) |
| 數量 | 至少 10 張(可包含人物剪影、手部、局部) |

## 🟢 加分(品牌建構)

### 8. 子品牌獨立 logo
| 子品牌 | 設計建議 |
|---|---|
| À Voir | 衣架線條 motif(極簡) |
| B Room | 妝刷或鏡框 motif |
| C Photos | 快門框或鏡頭 motif |
| | 不必複雜,但要讓客戶記住「這是 X 品牌」而不只是「字母圓圈」 |

### 9. 子品牌 accent color(可選)
- À Voir 偏褐(土地/秋季氣質)
- B Room 偏粉(柔和)
- C Photos 偏黑灰(精準)
- 跟主金色不衝突,各品牌頁面 hero 微調 accent

### 10. 共好發展總會 合作背書
- Logo(向量檔 SVG/AI)
- 合作描述一句話(例:「CHUN.EN 為共好發展總會合作之形象顧問」)
- 用途:首頁 hero 下方 trust signal 橫條

### 11. 合作企業 logo 牆(若有)
- B 端客戶常見:餐飲集團、保健品牌、藝文機構等
- 用途:首頁 trust signal 區塊

### 12. 媒體報導(若有)
- 截圖 + 連結
- 用途:首頁底部「Media」區塊

### 13. 客戶見證
| 規格 | 範例 |
|---|---|
| 格式 | 一段 quote(50-80 字)+ 客戶姓名 / 職稱 / 服務類型 |
| 範例 | 「轉職前找 CHUN.EN 重新規劃形象,同事第一天就稱讚我的氣質改變了。」<br>— 王小姐 / 科技業主管 / À Voir 形象規劃 |
| 數量 | 3-5 則 |

### 14. Pricing 起價帶(對 B 端客戶必要)
- 至少公開:「À Voir 形象診斷自 NTD X,XXX 起」「C Photos 個人形象照自 NTD X,XXX 起」
- 或公開共好套組價格(已是對外發行的行銷品)
- 用途:`packages.html`(已刪)若重啟,或加入 services 頁面

### 15. Cloudways 既有預約系統 URL
- 用途:`booking.html` 加「直接選日期預約」按鈕,讓熟客跳過 LINE 對話
- 對應你現有的 n8n + Supabase + LINE Bot 自動化

### 16. og:image 主圖
- 用途:Facebook / LINE 分享連結預覽
- 規格:1200×630px JPG(可由 dsc03134 logo 牆裁切)
- 目前用 dsc03269 暖光單椅作為臨時 og:image
